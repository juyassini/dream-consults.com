üîí SECURITY - Change Admin Password
-----------
For production, you MUST change the admin password:

1. Open Command Prompt
2. Run this to generate a secure hash:
   php -r "echo password_hash('your-new-password', PASSWORD_DEFAULT);"
3. Copy the long hash that appears
4. Open .env file and paste it:
   ADMIN_PASSWORD_HASH=<paste-hash-here>
5. Restart PHP server
